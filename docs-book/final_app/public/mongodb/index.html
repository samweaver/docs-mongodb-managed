This documentation describes the MongoDB Enterprise Service for Pivotal Cloud Foundry (PCF). The MongoDB Enterprise Service for PCF tile registers a service broker on PCF and exposes its service plans on the Marketplace. Those plans are integrated with MongoDB Ops Manager. Developers can provision MongoDB databases by creating instances of the service plans using Apps Manager or the Cloud Foundry Command Line Interface (cf CLI).

Overview

The MongoDB Enterprise Service for PCF has the following key features:

  • The ability to provision a MongoDB service instance in standalone, replica set, or sharded cluster configuration
  • Binding apps to the instance creates unique credentials for each app to access the database
  • Supports deployment of MongoDB Community or Enterprise server

Requirements

Before installing the MongoDB Enterprise Service for PCF tile, you must download and install a licensed copy of MongoDB Ops Manager. MongoDB Ops Manager deploys, manages, monitors, and backs-up MongoDB servers in your PCF and other environments.

Download MongoDB Ops Manager from MongoDB, and follow the MongoDB documentation to install it.

A prerequisite for the use of Ops Manager and MongoDB Enterprise Service for PCF in production is an Enterprise Advanced Subscription that covers all MongoDB instances. See the License section for more information.

Limitations

The MongoDB Enterprise Service for PCF has the following limitations:

  • Standalone and Replica set plans are pre-configured to one and three nodes respectively.
  • Only one VM
  • Support for creating blank MongoDB agent VMs is not available.

Feedback

Please provide any bugs, feature requests, or questions to the Pivotal Cloud Foundry Feedback list.

License

Before deploying MongoDB Enterprise Service for PCF in production, you must procure a MongoDB Enterprise Advanced Subscription for your deployment as well as MongoDB Professional Services.

MongoDB Enterprise Advanced Subscription includes a license to run MongoDB Ops Manager, which integrates with MongoDB Enterprise Service for PCF.

For more information about MongoDB Enterprise Advanced and Professional Services contact MongoDB.

You may use MongoDB Enterprise products in development without a subscription.
